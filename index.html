Module

Timecards

View, create, edit, audit, and manage every time record generated by your field devices. This is where daily auditing and payroll prep happens.

Overview — What Are Timecards?
How timecards are created, where to find them, and the two views

Timecards are records that capture every activity an employee performs — start time, end time, duration, Job, Site, Location, pay style, rate, Production Records, Crew, and Employer. Each time a worker is checked into an activity on a device, a Timecard is created automatically.

Where to Find Them

Navigate to Timecards from the left sidebar. The page defaults to today's date in List View. Use the date drop-down to select a different day, a range, or a preset like "Last 7 days." The arrow buttons step forward or back one day.

Two Views

List ViewTimeline View
Best forComplex edits, bulk changes, CSV exportVisual confirmation, quick drag-to-adjust edits
EditingSingle and bulk via flyout panelIndividual only — drag check-in/check-out edges
Unique featuresSortable columns, CSV export, extra hidden columnsColor-coded blocks Blue = Work Yellow = Break
Tip: Add the Reviewed, Device, and Geolocation columns in Settings → Column Visibility for a complete audit view.

Search & Filter

Click Search in the top header. Combine criteria and click Apply. Available filters: Name, Site/Location, Crop, Employer, Crew, Job, Device, Hourly Rate, Production Records, Work Hours, Audited by, Has no Prod. Records, Edited Record.

Check "Edited Record" to see only timecards modified after creation — useful for spotting what's already been touched.

Right-Click Context Menu

One selected: Navigate to Employee Profile · Edit · Duplicate · Transfer · Archive · Add New

Multiple selected: Edit Timecards · Switch Crew · Archive Timecards

Shift+Click selects a consecutive range. Ctrl+Click picks non-adjacent timecards.

Creating & Duplicating Timecards
Add missing records manually or copy from an existing employee

How to Create a Timecard

1
Click the + icon to the right of the search bar → Select Add timecards
2
Add employees by name, or switch to Crew to add an entire crew at once
3
Click + Add Timecard. Set start time, crew, site, location, work bundle, and job
4
Need multiple entries? Click + Add another timecard for each (different jobs, breaks)
5
Click Checkout to set end time → Save
Adding by crew is faster when everyone is missing — type the crew name and all members are added in one step.

Duplicate a Timecard

Copies all attributes from an existing timecard and assigns it to a different employee. Right-click → Duplicate Timecard. Change the employee name and any other details → Save.

You can assign the duplicate to multiple employees at once — add several names before saving.
Editing Timecards
Single and bulk editing — times, jobs, sites, pay styles, crew

Editing One Timecard

Select one timecard → right-click → Edit Timecard. The flyout shows all timecards for that employee on the selected day. Edit any attribute: start time, end time, crew, site, location, work bundle, job, pay style, rate.

Editing Multiple Timecards

Select multiple timecards (Shift+Click or Ctrl+Click) → right-click → Edit Timecards. Use checkboxes to apply bulk changes: Start Time, Checkout Time, Site & Location, Job, Pay Style & Rate, Duration.

Changes apply to all selected timecards. If times were changed, a confirmation screen shows before/after in timeline view. Review carefully before saving.

Pay Rate Rule

The higher rate always wins. An employee's unique compensation on their profile overrides the job's default rate. If no rate is set on the profile, the job's minimum wage applies.

Switching Crew

Select one or multiple timecards → right-click → Switch Crew → choose the correct crew → Save. Switch Crew works for both single and multiple selections.

Breaks & Lunches
Adding bulk breaks, fixing doubles, using the Exclude filter

Adding Breaks in Bulk

1
Filter to the crew. Select all employees missing breaks.
2
Click + → Add static duration jobs
3
Check "Dynamic start time after first checkin"
4
Set timing per your state's regulations (e.g., lunch 5 hours after check-in)
5
Select Break or Lunch. If both are missing, click "+ add another job"
6
Save
"Dynamic start time" means relative timing — give everyone a break X hours after their individual start time. No manual calculation needed even if employees started at different times.

The Exclude Filter — Critical Technique

Changing jobs? Always exclude breaks first. Otherwise the bulk edit overwrites break job data. Changing ranch/block? Include breaks so they match the correct location.
1
Open Search / Filter bar
2
Under Job, select Break and Lunch
3
Click "Exclude" instead of "Include" → Apply
4
Breaks are hidden. Select remaining timecards and bulk edit safely.
Daily Auditing Workflow
The 8-step daily audit and weekly payroll cycle

Daily Audit Walkthrough (Full Crew)

/videos/timecards/daily-audit.mp4

Daily Audit Steps

1
Set date and filter to one crew. Select yesterday. Open Search → Crew → select the crew → Apply. Never audit without filtering to a crew first.
2
Sort by Name, then Start Time. Groups each employee's timecards chronologically.
3
Correct flagged errors. Red icons appear automatically. Look for: missing check-ins, multiple check-outs, missing piece records, overlapping timecards, 24+ hour timecards.
4
Verify start times, end times, and total hours. Watch for 12:00 AM starts and checkout times past midnight.
5
Verify breaks and lunches. Switch to Timeline View — look for yellow blocks. No yellow = missing breaks.
6
Verify jobs, sites, and locations against crew timesheets.
7
Visual check with Timeline View. No gaps, no 24-hour blocks, yellow breaks visible for every employee.
8
Review and approve. Select all → right-click → Review Timecards. Run the Unreviewed filter as your final check.
By auditing daily, weekly payroll becomes a quick review rather than a scramble.
Errors & Fixes
Every common timecard error — symptoms, fix steps, and prevention

PickTrace flags errors with a red icon next to the employee name. Click any error below to expand.

Missing Checkout / Running Timecard
Most Common

Fix: Missing Checkout

/videos/timecards/errors/missing-checkout.mp4

Symptoms

  • 12:00 AM start time on the next day's card
  • ~24 hours shown
  • Timeline runs the entire day through 11:59 PM
  • Bulk checkout changes skip this employee

Fix

1
Verify actual checkout with your field team
2
Select the timecard → right-click → Edit
3
Update the End Time → Save
Bulk shortcut: Right-click → "Select all 12am start times" → Archive. Then set correct start time in bulk.
Wrong Date on Checkout
Common

Fix: Wrong Date on Bulk Checkout

/videos/timecards/errors/wrong-date.mp4

Bulk checkout saves to tomorrow because a running timecard pushed past midnight.

This is the #1 bulk-edit trap. Always check the date field before saving. If it shows tomorrow, change it to today. Root fix: always clear running timecards first.
Wrong Job
Common

Fix: Wrong Job on Timecards

2
Select timecards → right-click → Edit → check "Set Job"
3
Select correct Work Bundle and Job → Save
Jobs depend on the Work Bundle. If the job isn't in the drop-down, change the Work Bundle first.
Wrong Location
Common

Fix: Wrong Location

/videos/timecards/errors/wrong-location.mp4
1
Select timecard(s) → right-click → Edit
2
Check "Set Site and Location" → choose correct location → Save
Include breaks in your selection when changing location so they match the correct ranch/block.

What Are Production Records?
Where they come from and how to read the Production Records page

Production Records Overview

/videos/production/overview.mp4

Production Records are created on the handheld device when a crew leader or device user credits an employee with a unit of production (e.g., one bin, one tray, one tree). Each record is linked to an employee, a site, a location, a crop, a job, a unit count, and a piece rate.

Crew Scheduler

Schedule your crews to jobs, sites, and locations before the day starts. This is where piece rates, job assignments, and break prompts are all configured.

What Is the Crew Scheduler?
Why scheduling matters and what it controls in the field

Crew Scheduler Overview

/videos/crew-scheduler/overview.mp4

The Crew Scheduler tells PickTrace where each crew will work, what jobs they'll do, and what rates apply — before the field day begins. Device users in the field see only the jobs and locations that were scheduled for their crew that day, reducing errors.

The Crew Scheduler is the single most important setup step for accurate payroll. If a job isn't scheduled, it won't appear on the device, leading to wrong-job errors.

Locations & Piece Rates
Set per-block piece rates inside the scheduler

Setting Piece Rates in the Crew Scheduler

/videos/crew-scheduler/piece-rates.mp4

Piece rates are set per block within the Crew Scheduler. When you add a location to a schedule, you'll see a Piece Rate field for that block. Enter the dollar amount per unit (e.g., $2.00 per bin). This rate flows through to Production Records and ultimately to payroll.

PickTrace uses the highest rate that applies. Rate hierarchy: Employee Profile rate → Job rate → Crew Scheduler rate → minimum wage.
Break & Lunch Prompts
Configure when the device prompts crew leaders for breaks

Updating Break & Lunch Prompts

/videos/crew-scheduler/break-prompts.mp4

Break and lunch prompts are configured per crew. Go to Crews in the sidebar → select the crew → Edit → under "Break and Lunch Prompt" toggle on/off, set the timing, and add additional prompts as needed → Save.

If prompts fire at the wrong time, it's likely the timing is misconfigured at the crew level. Update each affected crew's settings.

Reports

Build custom reports, save templates, and export data to spreadsheets or your payroll system.

Step 1 — Choose Your Report Format
Daily vs. weekly, one crew vs. whole company, which template to start with

Introduction to Reports — Choosing a Format

/videos/reports/intro-format.mp4

Before building a report, answer two questions: What time range do I need? (a single day vs. the full week) and Who am I looking at? (one crew vs. the entire company). Your answers determine which report template and filters to use.

Recommended Starting Template: Employee Hours

The Employee Hours report is the best all-around starting point — it gives enough detail without overwhelming you. It provides one row per employee per date, broken out by site, and subtotals per employee so you can see totals at a glance.

Always use Employee ID (PT ID), not employee name, as your row identifier. If two employees share the same name, the report will merge their records together. Employee ID is always unique.

Rows & Sections

Under Rows & Sections in the report builder, you control how data is grouped. The default gives you a unique row per date, per site, and per employee. If you're running a single-crew report, you can remove the crew subtotal — it just adds an extra line you don't need when only one crew is visible.

Step 2 — Select & Customize Columns
Which columns matter, Total Hours vs. Paid Hours, and renaming columns

Selecting and Customizing Report Columns

You can rename any column header to match your payroll system's terminology. For example, rename "Employee ID" to "DataTech ID" or "ADP Number." Click the column name in the column builder to edit it.

Payroll

Map your PickTrace jobs, sites, and locations to payroll codes, then export the file directly into your payroll system (DataTech, ADP, Famous, and more).

How Payroll Works in PickTrace
PickTrace as the timekeeping system — not the payroll processor

Payroll Overview

/videos/payroll/overview.mp4

PickTrace collects your time, pay, and pieces. Your payroll system (DataTech, ADP, etc.) generates checks, pay stubs, and calculates withholding taxes. The two systems connect via an import file — you pull a CSV from PickTrace and import it into your payroll system.

PickTrace calculates: Hourly pay, piecework pay, overtime/doubletime, rest break premiums, sick pay, and minimum wage adjustments. Your payroll system handles withholding, check generation, and pay stubs.
Payroll Code Mapping
One-time setup: map PickTrace jobs/sites to your payroll codes

Mapping Payroll Codes (DataTech Example)

/videos/payroll/code-mapping.mp4

This is a one-time configuration. In your Payroll settings, map each Job, Site, and Location in PickTrace to the corresponding code in your payroll system. Once mapped, these codes are automatically included in every export.

PickTrace FieldDataTech Equivalent

EmployerEmployer code
Ask your payroll provider for an example export file if you're unsure how to map fields. This makes alignment much easier.

Pay Calculations
How PickTrace calculates hourly, piecework, overtime, and premiums

Understanding Pay Calculations

/videos/payroll/calculations.mp4
Pay TypeHow PickTrace Handles It
HourlyStart to end time × hourly rate
PieceworkUnit count × piece rate per unit
Hourly + PieceBoth calculated, employee receives higher of the two
Overtime / DoubletimeCalculated per state rules (daily or weekly)
Rest Break Premium1 hr × Regular Rate of Pay (weekly total ÷ hours worked)
Minimum WageApplies if piece pay falls below state minimum
The higher rate always wins. Employee profile rate → Job rate → Crew Scheduler rate → minimum wage.

Compliance Dashboard
Missed breaks, overtime, meal premium penalties, wage adjustments

Pulse Compliance Overview

/videos/pulse/compliance.mp4

The Compliance view surfaces potential violations automatically — missed meal periods that trigger premium penalties, employees approaching or exceeding overtime thresholds, and minimum wage adjustments. Review this daily alongside your timecard audit to catch issues before payroll is processed.

Meal Premium Penalty: When a break is missed and a penalty is applied, PickTrace adds 1.0 hour at the employee's Regular Rate of Pay (weekly total ÷ total hours worked that week) automatically. Use Reports → Column Visibility → add "Is Penalty Applied" and "Penalty Hours" to audit these.
