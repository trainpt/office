Timecards Guide
User Guide

Timecards

View, create, edit, audit, and manage timecards in PickTrace Office. Select a section above to get started.

Overview
What Are Timecards?

Timecards are records that capture every activity an employee performs — start time, end time, duration, Job, Site, Location, pay style, rate, Production Records, Crew, and Employer. Each time a worker is checked into an activity on a device, a Timecard is created automatically in PickTrace Office.

Where to Find Them

Navigate to Timecards from the left-hand sidebar under Dashboard. The page defaults to today's date and displays the List View.

Use the date drop-down to select a different day, a date range, or a preset like "Last 7 days". The arrow buttons step forward or back one day.

Records created on handheld devices appear here once the device syncs. The number of timecards per employee depends on how many activities and locations they worked.

Two Views
Timeline View vs List View

Toggle between both views using the buttons in the top-right corner of the page. Both let you review and audit timecards, but display information differently.

List ViewTimeline View
Best forComplex edits, bulk changes, investigation, CSV exportVisual confirmation, quick drag-to-adjust edits
EditingSingle and bulk via flyout panelIndividual only — drag check-in/check-out edges
Unique featuresSortable columns, extra columns (device, geolocation, reviewed), CSV exportColored time blocks, hover detail popup, visual gaps/overlaps

List View

The default view. Sortable table with all attributes as columns. Click any header to sort, drag to rearrange. Open Settings > Column Visibility to add or remove columns.

Add "Reviewed", "Device", and "Geolocation" columns for a complete audit view. Hidden by default.

Timeline View

Visual representation of the crew's day. Blue blocks are productive work, yellow are breaks. Drag edges to adjust times, hover for details, right-click for the full menu.

Search & Filter
Finding Timecards

Click Search in the top header next to the date to open the filter panel. Combine multiple criteria and click Apply. Available filters:

Name • Site/Location • Crop • Employer • Crew • Job • Device
Hourly Rate • Production Records • Work Hours • Audited by • Has no Prod. Records • Edited Record
Check "Edited Record" to show only timecards modified after creation — useful for seeing what's already been touched.
Quick Actions

The options change depending on how many timecards you have selected.

Selection Methods
Shift+Click — select a consecutive range (click first, hold Shift, click last).
Ctrl+Click — pick individual timecards that aren't next to each other.
Creating
How to Create a Timecard

Create timecards when an employee is missing a record — the device user forgot the device, didn't clock someone in, or the employee wasn't scanned. Works from either view.

1
Click the + icon to the right of the search bar. Select Add timecards.
2
Add employees by name, or switch to Crew to add an entire crew at once.
3
Click + Add Timecard. Set the start time, crew, site, location, work bundle, and job.
4
Need multiple entries (different jobs, breaks)? Click + Add another timecard for each.
5
Click Checkout to set the end time, then Save.
Adding by crew is faster when everyone is missing — type the crew name and all members are added in one step.
Duplicating

Duplicate a Timecard

Copies all attributes from an existing timecard and lets you assign it to a different employee. Right-click the timecard → Duplicate Timecard. A flyout shows the original attributes — change the employee name and any other details, then Save.

You can assign the duplicate to multiple employees at once — add several names before saving.
Transferring

Transfer a Timecard

Reassigns a timecard from one employee to another. The original employee loses the record entirely. Select one timecard → right-click → Transfer Timecard → update the employee name.

Transfer only works for a single timecard. To move multiple records, use Duplicate + Archive instead.
Editing
Single vs Bulk Editing

Editing One Timecard

Select one timecard → right-click → Edit Timecard. The flyout shows all timecards for that employee on the selected day. Edit any attribute: start time, end time, crew, site, location, work bundle, job, pay style, rate.

At the bottom of the flyout: Archive Timecard and Add New Timecard.

Editing Multiple Timecards

Select multiple timecards (Shift+Click or Ctrl+Click) → right-click → Edit Timecards. The flyout shows selected employees and checkbox fields for bulk changes:

  • Start Time
  • Checkout Time
  • Site & Location
  • Job
  • Pay Style & Rate
  • Duration
Changes apply to all selected timecards. If times were changed, a confirmation screen shows before/after in timeline view. Review carefully before saving.

Pay Rate Rule
The higher rate always wins. An employee's unique compensation on their profile overrides the job's default rate. If no rate is set on the profile, the job's minimum wage applies. Right-click any employee → navigate to their Employee Profile to view or update compensation.

Breaks & Lunches
Adding Breaks in Bulk

Breaks and lunches appear as yellow blocks in the Timeline view. When an entire crew is missing breaks — often because the field user skipped the prompt — add them for everyone at once.

1
Filter to the crew. Select all employees missing breaks.
2
Click the + icon next to the search bar → Add static duration jobs.
3
Check "Dynamic start time after first checkin".
4
Set the timing per your state's regulations (e.g., lunch 5 hours after checkin).
5
Select Break or Lunch. If both are missing, click "+ add another job".
6
Save.
"Dynamic start time" means relative timing — "give everyone a break X hours after their individual start time." No manual calculation needed even if employees started at different times.
Inserting

Adding a Break Mid-Day

When adding a break into the middle of an existing day, you must also add back the continuation timecard after the break. Otherwise the employee appears to stop working after their break.

Don't forget the continuation. Add the break, then create a new timecard that picks up immediately after the break ends, in the same job and location.
Common Fix

Double Break / Double Lunch

Happens when the crew leader taps the break prompt twice due to slow reception. Two consecutive break timecards appear with no work continuation after them.

1
Select the duplicate break → right-click → Archive check in
2
Select the timecard before the break → Duplicate check in
3
Edit the start time to begin immediately after the break → Save
Prevention: Tell field teams to tap the break prompt once only, even if the screen is slow.
Critical Technique
The Exclude Filter

When bulk-changing jobs, you must exclude breaks and lunches from your selection first. Otherwise the bulk edit overwrites break data.

1
Open the Search / Filter bar
2
Under Job, select Break and Lunch
3
Click "Exclude" instead of "Include" → Apply
4
Breaks are now hidden. Select remaining timecards and edit safely.
Changing jobs? Exclude breaks. Changing ranch/block? Include breaks so they match the correct location. This is the most important rule for safe bulk editing.
Configuration

Break & Lunch Prompts

Break/lunch prompts are configured per crew. If prompts fire at the wrong time, update them: Crews (left sidebar) → select the crew → Edit → under "Break and Lunch Prompt", toggle on/off, set the timing, and add additional prompts as needed. Save.

Daily Workflow
Daily Timecard Auditing

Each morning, audit yesterday's timecards for errors and correct them. Repeat for each crew.

1

Set date and filter to one crew

Select yesterday's date. Open Search → Crew → select the crew → Apply.

Never audit without filtering to a crew first. You risk editing another crew's data.
2

Sort by Name, then Start Time

Click the Name column header, then the Start Time header. This groups each employee's timecards chronologically.

3

Correct flagged errors

Red icons appear automatically next to employee names. Look for:

  • Missing check-ins — checkout without a preceding checkin
  • Multiple check-outs — two checkouts in a row
  • Missing piece records — "Missing" in the Piece column
  • Overlapping timecards
  • 24+ hour timecards (running from midnight)
4

Verify start times, end times, and total hours

Check for 12:00 AM starts (running timecards), checkout times past midnight, and totals that don't match expected shift length.

5

Verify breaks and lunches

In the Timeline view, look for yellow blocks. Their absence means breaks are missing. Use Add Static Duration to add them in bulk.

6

Verify jobs, sites, and locations

Compare PickTrace timecard details to the crew timesheets for accuracy.

7

Visual check with Timeline

Switch to the Timeline View for a final scan. No gaps, no 24-hour blocks, yellow breaks visible for every employee.

8

Review and approve

Select all timecards → right-click → Review Timecards. Run the Unreviewed filter as your final check.

By auditing daily, weekly payroll becomes a quick review rather than a scramble.
Weekly Workflow
Payroll Auditing Cycle

Follow this cycle each time you process payroll.

1

Crew leaders complete crew sheets throughout the day — hours, location, pieces, jobs.

2

Crew leaders submit sheets after shift or the next morning.

3

Payroll admin edits timecards in PickTrace against crew sheets.

4

Payroll admin calls crew leaders with questions from the audit.

5

End of week: payroll admin sends crew leaders a weekly summary.

6

Crew leaders validate totals, sign, and return.

7

Payroll admin makes final edits based on feedback.

8

Payroll is submitted.

Troubleshooting
Common Errors

PickTrace flags errors with a red icon next to the employee name. Click any error below for symptoms, fixes, and prevention.

Missing Checkout / Running Timecard Most Common
Employee never clocked out. System gives checkout at 11:59 PM.
▶
Symptoms
  • 12:00 AM start time on the next day's card
  • ~24 hours shown
  • Timeline runs the entire day through 11:59 PM
  • Bulk checkout changes skip this employee
Fix
  1. Verify actual checkout with your field team
  2. Select the timecard → right-click → Edit
  3. If multiple timecards: check "Set Checkout Time", update, Save
  4. If one timecard: edit the "End Time" directly, Save
Bulk shortcut: Right-click → "Select all 12am start times" → Archive. Then set correct start time in bulk.
Prevention

Train device users on checkout. Consider long-press checkout (without a badge) when the team is scattered.

Wrong Date on Checkout Common
Bulk checkout saves to tomorrow because a running timecard pushed past midnight.
▶
Symptoms
  • Timecard spans two calendar days
  • Total hours inflated
  • Date field shows tomorrow
Fix
  1. Before saving: check the date field
  2. If it shows tomorrow, change to today
  3. Root fix: always clear running timecards first
This is the #1 bulk-edit trap. Always check the date before saving.
Missing Check-In Common
Checkout exists without a preceding check-in.
▶
Fix
  1. Select the timecard → Edit
  2. Add a check-in point to complete start and end times
  3. Save
Multiple Check-Outs Common
Two checkouts in a row without a check-in between them.
▶
Fix
  1. Select the timecard → Edit
  2. Remove one of the checkouts, or add a check-in between them
  3. Save
Wrong Job Common
Device user forgot to switch jobs before scanning.
▶
Fix (Bulk)
  1. If breaks exist: use the Exclude filter first (Search → Job: Break/Lunch → Exclude → Apply)
  2. Select timecards → right-click → Edit → check "Set Job"
  3. Select correct Work Bundle and Job → Save
Fix (Single)
  1. Right-click → Edit → update Job in the flyout → Save
Set the correct pay style too when changing jobs. If switching hourly to piecework, check both "Set Job" and "Set Pay Style".
Prevention

Train device users to verify the job before scanning. Ensure jobs are scheduled via the Crew Scheduler.

Wrong Location Common
Device user forgot to switch location before scanning.
▶
Fix
  1. Select timecard(s) → right-click → Edit
  2. Check "Set Site and Location"
  3. Choose correct location → Save
Include breaks in your selection when changing location so they match the correct ranch/block.
Prevention

Train device users to verify Site and Location before scanning.

Overlapping Timecards Tricky
Two timecards overlap in time.
▶
Fix
  1. Select the timecard → Edit
  2. Review start/end times of each overlapping timecard
  3. Adjust so they don't overlap, or archive the incorrect one
Double Break / Double Lunch Preventable
Break prompt tapped twice. Two consecutive breaks, no continuation.
▶
Fix
  1. Archive the duplicate break
  2. Select the timecard before the break → Duplicate check in
  3. Set start time to immediately after the break → Save
Prevention: Tell field teams to tap the prompt once only.
Employee Missing from Crew Common
Employee worked but was never clocked in.
▶
Symptoms
  • Head count is off
  • No timecard exists for the employee
Fix
  1. Find a coworker who did the same work
  2. Right-click their timecard → Duplicate
  3. Assign to the missing employee, adjust details → Save
You can also create a timecard from scratch — see the Creating tab. Duplicating is just faster when a coworker did the same work.
Same Employee in Two Crews Rare
Badge scanned into two crews on the same day.
▶
Symptoms
  • Employee under two crew names on the same day
  • Same moment: system throws overlapping error
  • Different times: system allows it (crew change)
Fix
  1. Review both crews to find where the employee actually worked
  2. Archive the incorrect timecard
Missing Piece Record Warning
Piece/hourly+piece timecard with no production records. Shows "Missing" in the Piece column.
▶
Fix
  1. Open Production Records in a new tab
  2. Check if the employee should have a record
  3. If yes: add a production record
  4. If no pieces were logged: the warning can be ignored
This is a warning, not a hard error. It shows as text in the Piece column, not a red icon.
Archiving
Archive & Unarchive

Archiving removes a timecard from reporting but does not delete it. It can be restored at any time. Works from either view.

Archive

Select one or multiple timecards → right-click → Archive Timecard(s).

Or: open the Edit flyout and click "Archive Timecard" at the bottom of each card.

Unarchive

1
Click the settings gear (top-right) → toggle on "Show archived"
2
Select the archived timecard(s)
3
Right-click → Unarchive Timecard(s)
You only need "Show archived" when unarchiving. For archiving, the timecard is already visible.
Reviewing
Review & Approve Timecards

Reviewing marks timecards as audited and approved, clearing the "unreviewed" error. This signals the data is ready for payroll.

1
Select the timecard(s) to review
2
Right-click → Review Timecards
Use the Unreviewed filter as your final check. Search → check "Unreviewed" to find anything still needing attention before payroll export.
